 -d: run in disconnected mode (we can close terminal)
 --hostname: server name that Solace will have
 --name: docker container name (optionaly)
 -p: port 1883:1883 (1883 port for computer, 1883 port inside container)
 -p: port 9001:9001 (for queue - map  internal port to the external port (localhost - not external network) on the machine)
 -solace/solace-pubsub-standard - imaage to build or run this container (with management systems already installed in it)
